# terepla

*terepla* /ˈtɛɹəbla/ stands for **te**rminal **re**corder/**pla**yer.

The goal of this project is to
 * define the TRP file format,
 * provide a tool (`terepla`) to
   * record input to a terminal program and write it to a TRP file,
   * apply the contents of a TRP file to a terminal program.
